# EcoTrack - Carbon Footprint Tracker

A web app to track and reduce your daily carbon footprint. Built for the Kitchener-Waterloo-Cambridge community.

## Features

- **Dashboard** - Real-time stats, 7-day trend chart, category breakdown, and 3D Earth visualization
- **Log Activity** - One-click logging for transport, food, and energy activities
- **KWC Resources** - Local eco-friendly resources and comparison with regional average

## Tech Stack

- HTML5 / CSS3 / JavaScript (ES6+)
- Chart.js for data visualization
- Three.js for 3D Earth globe
- LocalStorage for data persistence

## Quick Start

1. Open `index.html` in a browser, or
2. Run a local server: `python -m http.server 8080`
3. Visit `http://localhost:8080`

## Demo

Press `Ctrl+D` to load sample data for demonstration.

## Deploy

Static site - deploy to Vercel, Netlify, or GitHub Pages.
